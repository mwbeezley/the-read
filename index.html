How Parlay Lab Works
Everything you need to build smarter parlays — from game selection to bet tracking.
1 — Start from the Dashboard
Pick a sport on the Home tab. The current week's schedule loads automatically from ESPN with spreads, totals, and records. Click Scout on any game to get deep AI research, Build to start adding legs manually, or Auto SGP to generate AI-recommended parlays instantly — zero manual entry needed.
2 — Scout a Game
The Scout tab is the research engine. Select a game, hit Scout, and Claude searches for live data — injuries, stats, trends, line movement, weather — then delivers a structured analysis: game script projection, key matchups, situational factors, and 3 recommended SGPs (conservative, balanced, aggressive). Each recommendation includes correlation reasoning and a one-click import to Builder. You can also ask follow-up questions in the Quick Research sidebar.
3 — Build Your Own Parlay
The Builder tab is your workbench. Add legs manually or generate them with AI. Set your platform (DraftKings / FanDuel), adjust wager and boost, see live correlation scoring and payout calculations. Hit Analyze to get Claude's sharp assessment of your parlay — correlation conflicts, matchup edges, X-factors, and a confidence rating. The AI uses web search for real-time context.
4 — Upload a Bet Slip
Already placed a bet on DraftKings or FanDuel? Drop a screenshot in the Upload tab. Claude reads the image and extracts every leg, the odds, wager, boost, and win/loss status automatically. From there: import to Builder to analyze, save to Pending/Placed for tracking, or hit Suggest Alternative to get a smarter version of the same parlay.
5 — Track & Learn
Pending holds bets you're considering or waiting on results. Mark them Win/Loss and they move to Placed, where your full record, P&L, and ROI are tracked. Once you have 5+ settled bets, the Pattern Analysis engine identifies what prop types, sports, leg counts, and situations have been hitting vs busting — so the AI can weight future recommendations toward what works for you.
6 — Library & Tools
Leg Library saves individual props you like for reuse across parlays. Tools includes the Parlay Grader (paste final stats to see where each leg landed), Alt Line Optimizer (find better lines), Sharp Money Detector, Correlation Matrix, and Fade Mode (contrarian analysis).
Quick Workflows
Full research play: Home → Pick game → Scout → Import rec → Adjust → Save to Pending
Quick auto play: Home → Auto SGP button → Review → Import → Place bet
Assess an existing bet: Upload tab → Drop screenshot → Analyze or Suggest Alternative
Track and learn: Pending → Mark outcomes → Placed → Pattern Analysis
Ad-hoc question: Scout sidebar → Quick Research → Ask anything about a matchup
Parlay Lab uses Claude AI with web search for real-time analysis. All data is stored locally in your browser. No account needed.
